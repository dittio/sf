Welcome to the Storyboard/Scene Microservices

We currently support the following calls:

  1. / - gives this 'help' page
  2. /story/list - gives a list of all stories
  3. /story/<id> - gives the details of the storyboard with <id>. E.g.:
    http://localhost:3000/story/1 - to show the details of the first story in our database.
  4. /story/add/name/description - adds a story to the database with the given name and description. The URL should be properly escaped - i.e. the spaces should be replaced with '+' etc. E.g.:
    http://localhost:3000/story/add/Test+Storyboard/This+is+our+first+storyboard to add a storyboard.
  5. /story/update/id/name/description - updates the story specified by id, with the given name and description. E.g.
    http://localhost:3000/story/update/1/First+Storyboard/This+is+our+first+storyboard" updates storyboard 1 with the new name and/or description.
  6. /scenes/list/for/story/<id> - gives all the scenes for storyboard with <id>. E.g.:
    http://localhost:3000/scenes/list/for/story/1 gives a list of all scenes for story 1.
  7. /scene/add/<sid>/<name>/<desc> - adds a scene to story <sid>, with specified name and description. E.g.
    http://localhost:3000/scene/add/1/First+scene/This+is+a+test+scene+for+our+first+storyboard adds a scene to the first storyboard.
  8. /scene/delete/<id> - deletes scene <id>. E.g.
    http://localhost:3000/scene/delete/1 should delete scene 1.
  9. /story/delete/<id> - deletes storyboard <id> and all scenes associated with it.
